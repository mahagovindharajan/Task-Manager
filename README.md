# 📋 TaskFlow – Task Manager

A modern and responsive task management web application designed to help users efficiently organize, track, and manage daily tasks.

---

## 🚀 Overview
TaskFlow is a frontend-based task manager built using HTML, CSS, and JavaScript. It provides a clean and intuitive interface for managing tasks with features like priority selection, filtering, and real-time updates.

---

## ✨ Features
- Add, delete, and complete tasks  
- Set task priority (Low, Medium, High)  
- Filter tasks (All / Pending / Completed)  
- Task statistics (Total, Completed, Pending)  
- Clear completed tasks  
- Real-time UI updates  
- LocalStorage support (persistent data)  
- Responsive and modern UI  

---

## 🛠️ Tech Stack
- **HTML5** – Structure  
- **CSS3** – Styling & layout  
- **JavaScript (ES6)** – Functionality & logic  

---

## ⚙️ How It Works
1. Enter a task in the input field  
2. Choose a priority level  
3. Click **Add Task**  
4. Mark tasks as completed when done  
5. Use filters to manage task visibility  
6. Clear completed tasks when needed  

---

## 📌 Key Highlights
- Efficient state management using JavaScript  
- Persistent data using browser LocalStorage  
- Clean UI with smooth user experience  
- Organized and scalable code structure
